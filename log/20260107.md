`React`는 `Componenet` 단위로 관리되어지는데 이 또한 계층 구조로 설명이 가능하다
이때 가장 상위 `component`를 보통 `App.jsx`로 지정하고 `root component`로 부른다
보통 `root component`를 제외한 다른 `component`들은 `componenets`폴더를 만들고 그 안에 만든다
구조는 다름과 같다
```jsx
//fuction Component() { 둘 중 편한거로
const Component = () = {
	return (
		<h1>aaa</h1>
	);
}
export default <Component>;
```
이걸 `Main.jsx`에서 불러와서 활용할 수 있다
```jsx
import <Component> from "Component path";
```
